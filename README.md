## ğŸ§ª How Nutrient Density is Evaluated

Nutrient density models typically score foods by comparing essential nutrients relative to energy content. For example, tools like the **NRF9.3 index** rank foods based on beneficial nutrients (e.g., fiber, vitamins) minus limiting nutrients (e.g., saturated fat, sugar). ([PubMed][1])
